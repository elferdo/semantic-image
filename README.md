# About

[In progress]

I have a large collection of pictures that I host locally because I
don't want to depend on cloud providers to manage them. Options for
semantic search on linux are limited, so I set to implement my own
local semantic search engine running local LLMs.

Make sure to have [Ollama](https://ollama.com/) installed and running
and a model that can work with images available, for example
[Gemma3](https://ollama.com/library/gemma3).
